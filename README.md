# customer-support-dashboard
Teljes értékű **PHP + MySQL + Bootstrap** ügyfélszolgálati rendszer ticketek kezelésére.

# Ügyfélszolgálati Dashboard

## Funkciók
- Új ticket nyitása
- Státusz váltás: új → folyamatban → kész
- Reszponzív dizájn, hover effektek, árnyékok
- MySQL adatbázis

## Tech Stack
- PHP
- MySQL (XAMPP)
- Bootstrap 5
- Custom CSS (hover + shadow)

## Telepítés
1. XAMPP → Apache + MySQL indítás
2. phpMyAdmin → `database.sql` import
3. Mappa: `htdocs/customer-support-dashboard`
4. Nyisd meg: `http://localhost/customer-support-dashboard`
